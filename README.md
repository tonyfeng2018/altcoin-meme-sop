# 山寨币/Meme SOP 极速排雷与动能评分

## 核心流程

```
步骤零：接收信息（Ticker + Chain + 合约地址）
    ↓
步骤一：一票否决排雷（Kill Switch）
    ↓ 通过 → 步骤二：识别资产属性 → 步骤三：打分
    ↓ 否决 → 输出🔴极度高危，终止
    ↓
步骤四：输出信号与投研建议
```

## 评分结构（满分100分）

| 模块 | 权重 | 说明 |
|------|------|------|
| 模块一：全局Beta | 20分 | BTC.D + TOTAL3大盘资金溢出 |
| 模块二A：常规模型 | 80分 | 适用流通率15-80%的叙事币 |
| 模块二B：高流通/Meme | 80分 | 适用流通率>90%的Meme/土狗 |

## 信号阈值

| 分数 | 信号 |
|------|------|
| ≥75分 | 🟢 强烈建议买入/建仓 |
| 55-74分 | ⚪ 底仓观望/等待突破 |
| ≤40分 | 🔴 规避/减仓/潜在做空 |

> 🚨 **排雷优先**：任何打分前必须先通过4条红线（老鼠仓、流动性、蜜罐貔貅、宏观熔断）
